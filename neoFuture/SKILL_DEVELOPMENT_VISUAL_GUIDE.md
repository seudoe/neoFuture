## 🔄 User Interactions

```
Video Search:
1. User types query → Click Search
2. Loading spinner appears
3. Videos display in grid
4. Click "Watch Video" → Opens YouTube

Voice Search:
1. Click microphone icon 🎤
2. Microphone turns red (listening)
3. Speak query
4. Query appears in search box
5. Auto-search or click Search

Government Schemes:
1. Scroll to schemes section
2. Read scheme details
3. Click "Register / Learn More"
4. Opens official website in new tab

Training Centers:
1. View all centers
2. Click filter (KVK/RSETI)
3. View filtered results
4. Click "Get Directions"
5. Opens Google Maps in new tab
```

## 🎨 Responsive Breakpoints

```
Mobile (< 768px):
├─ 1 column layout
├─ Stacked navigation
├─ Full-width cards
└─ Larger touch targets

Tablet (768px - 1024px):
├─ 2 column layout
├─ Horizontal navigation
├─ Medium cards
└─ Balanced spacing

Desktop (> 1024px):
├─ 3 column layout (videos)
├─ 2 column layout (schemes/centers)
├─ Sidebar navigation
└─ Optimal spacing
```
